* `DayExercise` 테이블 속성 수정 후 테스트

```sql
CREATE TABLE DayExercise (
    year VARCHAR2(10),
    month VARCHAR2(10),
    exdate VARCHAR2(10),
    day VARCHAR2(10),
    exName VARCHAR2(30),
    count NUMBER(10)
);
```
